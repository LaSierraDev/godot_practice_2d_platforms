# 🎮 Godot Practice — 2D Platformer

A small practice project built in **Godot 4.6.1**, focused on developing a **basic 2D platformer**.

## 🧩 Implemented Features

### 🎮 Gameplay
- Horizontal movement  
- Jump and gravity  
  + Coyote Time  
- Moving platforms  
- Traps  
- Collectibles  

### 🧱 Physics & Collisions
- Physics body and area handling  
- One-way collisions  
- Group system usage  
- Vector-based movement logic for player tracking  

### 👾 Enemies
- Automatic patrol movement with direction changes on collision  
- Enemy-to-enemy interactions  
- Player detection and chase behavior  

### 🖥️ UI & Scenes
- Score system  
- Options menu navigation  
- Scene switching  
- Node instancing  
- Node removal  
- `CanvasLayer` for proper UI layout  
- Responsive scaling for multiple resolutions  
- Custom UI themes  

### 🎨 Environment
- `TileMap`  
- Parallax background  
- Audio system  
- On-screen node activation/deactivation optimization  
- `AnimationPlayer`  

### 💻 Code Architecture (GDScript)
- Clean, readable, and scalable code  
- Modular structure with reusable components  
- Clear separation of responsibilities  
- Global variable management  

+ Patterns:
  + Signal Hub  
  + Singleton (Autoloads)  

- Signals  
- Custom signals  
- Timers  
- Tweens  
- Inherited scenes  

### 🎥 Camera
- Player follow system  
- Smooth movement  

## 🛠️ Technologies
- Godot Engine 4.6.1  
- GDScript  
- VSCode  
- Git (Version control)
