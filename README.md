# 🎮 Godot Practice — 2D Platformer

Small practice project developed with ***Godot 4.6.1*** building a **basic 2D platformer**.

## 🧩 Implemented Features 🧩 ##

### 🎮 Gameplay
- Side movement
- Jump and gravity
	+ Coyote Time
- Moving platforms
- Traps
- Collectibles

### 🧱 Physics & Collisions
- Handling of bodies and areas
- One Way Collision
- Groups usage
- Vector-based movement logic to follow the player

### 👾 Enemies
- Automatic patrol movement with direction change on collision
- Enemy-to-enemy interaction
- Player detection and chase behavior

### 🖥️ UI & Scenes
- Score system
- Options menu navigation
- Scene switching
- Node instancing
- Node deletion
- `CanvasLayer` for proper UI layout
- Responsive scaling for multiple resolutions
- Custom UI Themes

### 🎨 Environment
- `TileMap`
- `Parallax` background
- Audio system
- On-screen node activation/deactivation optimization
- `AnimationPlayer`

### 💻 Code Architecture (GDScript)
- Clean, readable and scalable code
- Modular structure and reusable components
- Separation of responsibilities
- Global variables management

+ Patterns:
  + Signal Hub
  + Singleton (Autoloads)

- Signals
- Custom Signals
- Timers
- Tweens
- Inherited scenes

### 🎥 Camera
- Player follow system
- Smooth movement

## 🛠️ Technologies
